# 🎬 Movie Rental Business Analysis (SQL Project)

## 📌 Overview
This project analyzes a movie rental business using the Sakila database. The goal is to extract insights related to customer behavior, revenue trends, and film performance.

## 🛠 Tools Used
- SQL (MySQL)
- Sakila Database

## 📊 Key Analysis
- Customer segmentation based on spending
- Monthly revenue trends
- Top rented films and categories
- Top 3 films in each category using window functions

## 📁 Project Structure
- dataset/ → database files
- queries/ → SQL queries
- outputs/ → screenshots of results
